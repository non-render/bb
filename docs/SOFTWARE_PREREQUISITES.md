# Software Prerequisites
- Atleast Windows 10 `14393.0` or Windows 11 `22000.100`
- .NET Desktop Runtime 8.0.11
  - https://dotnet.microsoft.com/en-us/download/dotnet/8.0
- A Microsoft account with Minecraft for Windows 10 owned or a valid game pass subscription.

## Developer Mode 
Developer Mode must be enabled. This ensures that the app can be run without requiring admin permissions (for example, a child or less-privileged account can run instead of needing admin each time). To enable Windows Developer Mode:

1. Open Windows Settings
2. Go to **Privacy & security** → **For developers**
3. Turn on **Developer Mode**

If Developer Mode is not enabled, the launcher will not start.
